# IntentTrace

IntentTrace is a prototype operational observability framework for studying distributed personalization systems.

## Overview

Modern personalization systems rely on distributed SDKs, context services, telemetry pipelines, and storage systems. IntentTrace provides a research and teaching environment for exploring operational behaviors such as:

- Timeout propagation
- Circuit-breaker interactions
- Telemetry overhead
- Cardinality explosions
- Batch and online workload interference
